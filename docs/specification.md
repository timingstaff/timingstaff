# Mission

라이브바 운영을 더 단순하게, 더 정확하게, 더 따뜻하게 만든다.

---

# Vision

운영의 모든 기록이 데이터가 되고,

데이터가 더 나은 운영을 만든다.

궁극적으로는 운영자의 경험을 시스템으로 축적하여,
누구나 안정적이고 일관된 운영을 할 수 있도록 돕는다.

---

# Core Principles

1. 기록은 기억을 대신한다.

2. 모든 기록은 운영을 개선하기 위해 존재한다.

3. 모든 사람은 자신에게 필요한 정보만 본다.

4. AI는 사람을 대신하지 않고 더 나은 판단을 돕는다.

5. 복잡함보다 직관을 우선한다.

6. 운영은 감이 아닌 데이터로 성장한다.

7. 시스템은 사람을 통제하기 위해 존재하지 않는다.
사람이 더 편하게 일하도록 돕기 위해 존재한다.

8. 반복되는 업무는 자동화하고,
사람은 더 중요한 일에 집중한다.

---

# 소개

TimingStaff는 라이브바 운영에 필요한 모든 업무를 하나의 시스템에서 관리하기 위해 개발 중인 운영 플랫폼입니다.

직원과 관리자가 같은 데이터를 기반으로 업무를 수행하고,

반복되는 업무는 자동화하며,

축적된 데이터를 AI가 분석하여 더 나은 운영을 지원하는 것을 목표로 합니다.

TimingStaff는 단순한 직원관리 앱이 아닙니다.

**라이브바 운영을 위한 Operating System(OS)** 을 만드는 프로젝트입니다.

# 개발 원칙

- 기능보다 운영을 우선한다.
- 운영보다 사용성을 우선한다.
- 모든 중요한 행동은 기록한다.
- 데이터는 시간이 지날수록 가치가 커진다.
- AI는 근거를 제공하고 결정은 사람이 한다.
- 실제 라이브바 운영을 기준으로 개발한다.
- 새로운 기능은 기존 철학과 충돌하지 않아야 한다.
- 하나의 기능은 하나의 목적만 가진다.
- 기록은 삭제보다 이력을 우선한다.
- 모든 화면은 10초 안에 이해되어야 한다.
