> 기록은 기억을 지배한다.

---

# 목적

TimingStaff를 사용하는 모든 사용자의 업무 흐름을 정의한다.

기능보다 사용자의 흐름을 우선하며,

모든 기능은 이 Workflow를 기준으로 개발한다.

# Workflow 원칙

- 모든 업무는 기록으로 남긴다.
- 중요한 공지는 반드시 확인 후 업무를 시작한다.
- 완료와 재확인은 구분한다.
- 모든 변경 사항은 작성자와 시간을 기록한다.
- AI는 추천과 분석만 제공하며, 최종 판단은 사람이 한다.
- Workflow는 실제 라이브바 운영을 기준으로 지속적으로 개선한다.
