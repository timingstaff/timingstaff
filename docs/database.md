# Database 원칙

- 하나의 데이터는 하나의 위치에서만 관리한다.
- 모든 변경은 작성자와 시간을 기록한다.
- 삭제보다 이력 보존을 우선한다.
- 관리자 데이터와 직원 데이터를 명확히 분리한다.
- AI는 저장된 데이터를 기반으로만 분석한다.
- 데이터 구조는 확장 가능하도록 설계한다.
