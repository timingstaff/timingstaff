# 👑 TimingStaff Admin Dashboard

> 사장은 문제부터 본다.

---

# 목적

관리자 대시보드는 사장님이 매장 전체 상태를 5초 안에 파악하기 위한 화면이다.

직원 화면은 “내가 해야 할 일” 중심이고,  
관리자 화면은 “지금 매장에 무슨 일이 있는가” 중심이다.

---

# 1. 관리자 화면 핵심 원칙

- 문제를 먼저 보여준다.
- 숫자보다 판단이 먼저다.
- 사장은 버튼을 많이 누르지 않아야 한다.
- 매출, 직원, 공지, 업무, 재고, 고양이를 한 화면에서 요약한다.
- AI는 결론이 아니라 근거와 추천을 제공한다.

---

# 2. 첫 화면 구조

## 2.1 Morning Brief

- 날짜
- 요일
- 날씨
- 영업 상태
- 오늘 공연 여부
- 예약 수
- 예상 매출

---

## 2.2 오늘 반드시 확인

가장 위에 표시한다.

예시:

- 공지 미확인 직원
- 미출근 직원
- 재고 긴급 항목
- 고양이 약 투약
- 공연 준비
- POS 마감
- 현금함 확인

---

## 2.3 오늘 운영 요약

- 직원 출근 현황
- 공지 확인률
- 업무 완료율
- 매출 달성률
- 재고 긴급 항목
- 고양이 케어 완료율

---

# 3. 관리자 대시보드 카드

## 3.1 직원 카드

- 출근 인원
- 미출근 인원
- 휴무 인원
- 마지막 활동 시간

---

## 3.2 업무 카드

- 전체 업무 수
- 완료 업무 수
- 미완료 업무 수
- 재확인 요청 수

---

## 3.3 매출 카드

- 현재 매출
- 목표 매출
- 달성률
- 목표까지 남은 금액
- 전일 대비
- 주간 대비

---

## 3.4 재고 카드

- 긴급 재고
- 확인 필요 재고
- 마지막 확인자
- 마지막 확인 시간

---

## 3.5 고양이 케어 카드

- 사료
- 물

- 약

---

## 3.6 공지 카드

- 전체 직원 수

- 미확인 직원
- 확인 시간

---

# 4. AI 운영 브리핑

AI는 현재 데이터를 기반으로 운영 브리핑을 생성한다.

예시:

- 오늘 업무가 3개 남아 있습니다.
- 공지 미확인 직원이 2명 있습니다.
- 레몬 재고 확인이 필요합니다.
- 현재 매출은 목표 대비 65%입니다.
- 더운 날씨로 생맥주와 얼음 소진이 빨라질 수 있습니다.

---

# 5. 타임라인

오늘 하루의 모든 주요 행동을 시간순으로 기록한다.

예시:

- 08:51 마린 출근
- 09:02 마린 공지 확인
- 09:10 주류 재고 체크 완료
- 09:20 고양이방 소독 완료
- 10:05 레몬 재확인 요청

---

# 6. 관리자 화면 우선순위

1. 오늘 반드시 확인
2. AI 운영 브리핑
3. 운영 요약
4. 직원 현황
5. 업무 현황
6. 매출 현황
7. 재고 현황
8. 고양이 케어
9. 타임라인

---

# 7. 향후 기능

- 관리자 전용 로그인
- 관리자 비밀번호
- 생체 인증
- 공지 작성
- 업무 추가/수정/삭제
- 재고 추가/수정/삭제
- 직원 추가/수정
- 매출 입력
- 공연 일정 관리
- AI 추천 업무
- AI 발주 추천
